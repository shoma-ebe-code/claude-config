このセッションで得た情報を整理して保存してください。

## 保存する内容

1. **解決した問題・ハマりポイント**（あれば）
2. **うまくいった手順・コマンド**（あれば）
3. **次回セッションへの引き継ぎ事項**（あれば）

## 保存先ルール

**技術的な学び・ナレッジ**がある場合:
→ `05-ai-dev-knowledge/learnings-YYYY-MM-DD.md` に保存（日付は今日）
→ すでに同日のファイルがあれば追記する

**次回への引き継ぎ**がある場合:
→ `CLAUDE.local.md` の「セッション間の引き継ぎメモ」セクションを更新する
→ 完了済みの事項は削除し、持ち越す事項だけ残す

## 出力形式

```markdown
## [日付] セッション学習記録

### 解決した問題
- ...

### うまくいった手順
- ...

### 参考リンク
- ...
```

保存後、何をどこに保存したかを1〜2行で報告してください。
